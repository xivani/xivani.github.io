Submit A New Case:

The File Room Submit a new case Interface is were users go to report incidents of censorship. Users fill out information from the date & location of incident, the reason for censorship along with were it took place more descriptive details on the incident this is than added to the archives. The different images put up of the interface is to give a sense of both how the interface looks and interacted with and how descriptive submitting a report is.
